iBOL BOLD (http://www.boldsystems.org)

This integration is in testing mode and will evolve according to user requests

Use case

Manage field and lab samples in EarthCape database, send samples to BOLD, download BOLD data back to the research database

API

http://www.boldsystems.org/index.php/API_Public/combined

Fields

  • DnaExtract.BoldBin
  • DnaExtract.BoldRecordId
  • Attachment.BoldMediaId
  • Attachment.BoldTraceFileId

Actions

  • DnaExtract -> Download BOLD (downloads BOLD record information if BOLDRecordId is present. Sequence and records are attached to DnaExtract)

Fields currently downloaded

  • DnaExtract.BoldRecordId API PARAM
  • DnaExtract.BoldBin (bin_uri)
  • DnaExtract.Unit.TaxonomicName
  • Attachment.BoldTraceFileId (trace_id) KEY
  • Attachment.ID (trace_id)
  • Attachment.Name (trace_name)
  • Attachment.BoldMediaId
  • Attachment.FileUrl (trace_link)
  • Attachment.Date (run_date)
  • Sequence.Name (sequenceID) KEY
  • Sequence.Marker (markercode)
  • Sequence.Nucleotides (nucleotides)
  • Sequence.DnaExtract
  • Sequence.Unit
  • Sequence.Genbank (genbank_accession)
  • Marker.Name (markercode)
