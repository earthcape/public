To modify a role, create a new one or add users to an existing role, go to Administration -> Roles.

To add new users to an existing role, open desired role. Users linked to this role can be found in Users sheet.

To add new users press Link button in the toolbar and select the users that you want to add to this role.

Save the changes.

Modify the role's properties by navigating to the Type Permissions sheet.

Modify the existing Object permissions or add a new object and modify its permissions.

Save the changes.

To create new role click New in the Roles window.

Select the Role Type and give your new role a Name. Add Users to the role.

Add Type Permissions as above.
