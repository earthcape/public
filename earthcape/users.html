Go to Administration -> Users, click New Type a User Name and click Link button in Roles window. Assign at least one role to user. By default the user has an empty password. To reset user password or assign a specific password to user:

  1. Right-click a user name and select Reset password.
  2. Copy the suggested password or type a new one in the field and click Reset password. This new password can now be used to log in once to program and it will require the user to change the password. If you want to continue using the assigned password, before exiting the program double-click the user name and deselect Change password on next logon.
  3. To change your own password open My details in the Main view. Click Change My Password in the toolbar.

Note! You have to belong to administrator group to be able to reset other users passwords.
